# Backend Portfolio Clone

This is a Next.js + Tailwind CSS portfolio inspired by `https://ceo.pronexus.in/`, rebuilt as a data-driven site for a senior backend engineer.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Edit all content in one place

Everything is driven from:

`src/data/portfolio.json`

That file controls:

- Personal info and hero copy
- Work/projects
- Blog posts
- Contact links
- Uses page content
- Resume PDF path
- Footer navigation

## Useful scripts

```bash
npm run dev
```
